```bash
cd ~/Aba/cafe/my-app
```


## 💡 Полезные советы

### **Offline режим**
1. Откройте приложение на планшете
2. DevTools → Network tab → Offline
3. Все видео работают без интернета ✅

### **Админ-панель**
1. Нажмите ТРИ раза быстро на экран
2. Откроется админ-панель
3. Выберите нужное действие

### **Режимы работы**
- **"Меню"**: Официант ведет заказ (корзина как блокнот)
- **"Интерактивный стол"**: Гость сам оформляет заказ (кнопка "Заказать")

## 🐛 Если что-то не работает

### **Видео не загружается**
```
1. Проверьте наличие файлов в public/media/
2. Консоль (F12) → Network tab → видео файл
3. Если 404 → проверьте путь в page.tsx
```

### **Service Worker не кэширует**
```
1. Откройте DevTools → Application → Service Workers
2. Проверьте статус "activated"
3. Cache Storage → cafe-menu-v1 → Проверьте файлы
```

### **Модальное окно не открывается**
```
1. Проверьте консоль на ошибки
2. Нажмите на иконку ℹ на карточке
3. Если не работает → проверьте структуру данных MenuCard
```
