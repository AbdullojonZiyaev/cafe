## 📋 Что было добавлено

### ✅ **Service Worker и Offline-доступ**
- Файл: `public/sw.js`
- Cache-First стратегия для видео и меню
- Автоматическая регистрация при загрузке приложения
- Полная работа без интернета после первого посещения

### ✅ **PWA Манифест**
- Файл: `public/manifest.json`
- Конфигурация для установки на планшеты (iOS/Android)
- Display: fullscreen для Kiosk Mode
- Встроенные иконки и метаданные

### ✅ **Service Worker Регистрация**
- Файл: `src/app/ServiceWorkerRegister.tsx`
- Автоматическая регистрация при монтировании
- Обработка ошибок и логирование

### ✅ **Модальное окно с информацией**
- Открывается при клике на иконку ℹ
- Показывает полную информацию о блюде:
  - Описание
  - Выход (вес) в граммах
  - КБЖУ (Калории, Белки, Жиры, Углеводы)
  - Аллергены с предупреждением ⚠️
- Кнопка "В корзину" прямо из модального окна

### ✅ **Два режима работы**
1. **Режим "Меню"** (по умолчанию)
   - Корзина видна как "блокнот"
   - Нет кнопки "Заказать"
   - Официант ведет заказ

2. **Режим "Интерактивный стол"**
   - Кнопка "Заказать" активна
   - Гость самостоятельно оформляет заказ

### ✅ **Админ-панель (скрытая)**
- Открывается при **тройном быстром нажатии** на экран
- Функции:
  - Включение/выключение Kiosk Mode (Fullscreen)
  - Переключение режимов работы
  - Быстрый выход

### ✅ **Уведомление о заказе**
- "Ваш заказ принят! Официант скоро подойдет."
- Автоскрытие через 2.5 сек
- Плавная анимация входа/выхода

### ✅ **Kiosk Mode поддержка**
- Fullscreen API для полноэкранного режима
- Управление через админ-панель
- Готовность для ресторана

### ✅ **CSS анимации**
- Плавный вход модального окна
- Анимация уведомления
- Слайд-ин эффекты

## 🎮 Как использовать новые функции

### **Добавление товара в корзину**
```
1. Нажмите на кнопку "В корзину" на карточке
   или
2. Откройте модальное окно (иконка ℹ) и нажмите там
```

### **Просмотр деталей блюда**
```
1. Нажмите на иконку ℹ на карточке
2. Просмотрите описание, КБЖУ и аллергены
3. Закройте крестиком или кликом вне окна
```

### **Оформление заказа (режим "Интерактивный стол")**
```
1. Откройте корзину (нажмите на заголовок)
2. Нажмите кнопку "Заказать"
3. Получите подтверждение: "Ваш заказ принят!"
4. Корзина очистится автоматически
```

### **Включение Kiosk Mode (Администратор)**
```
1. Нажмите ТРИ раза быстро на экран
2. Откроется админ-панель
3. Нажмите "Киоск-режим (Fullscreen)"
4. Приложение перейдет в полноэкранный режим
```

### **Переключение режимов (Администратор)**
```
1. Нажмите ТРИ раза на экран для открытия админ-панели
2. Нажмите "Режим: [текущий режим]"
3. Режим переключится и сохранится в localStorage
```

## 🧪 Тестирование

### **На компьютере (DevTools)**
```bash
1. npm run dev
2. Откройте http://localhost:3000
3. F12 → Device Toolbar → iPad Pro (1024x1366)
```

### **На реальном планшете**
```bash
1. npm run build
2. npm start
3. Откройте http://[IP]:3000 на планшете
4. Safari (iOS): Поделиться → На экран Домой
5. Chrome (Android): Меню → Установить приложение
```

### **Проверка offline режима**
```
1. Откройте приложение на планшете
2. DevTools → Network → Offline
3. Все видео и меню работают без интернета
```
